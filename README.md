# java-url-converter
Online Converter for Java SE Download URLs for Java 8 or older

> [!NOTE]
> It may contain some AI-processed code.
> Its content has been manually reviewed and, where necessary, tested.

> [!IMPORTANT]
> It is possible that the converter will lose its functionality over time if the owner of the links to be converted changes the method of provision.
> 
> **Therefore, there is no guarantee of functionality over a longer period of time.**
